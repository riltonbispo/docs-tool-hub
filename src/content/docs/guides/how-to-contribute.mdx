Para contribuir com o nosso projeto, siga os passos abaixo:

<Steps>

1. **Faça um Fork do Repositório**
   - Vá para a página principal do repositório no [GitHub.](https://github.com/riltonbispo/tool-hub)
   - Clique no botão "Fork" no canto superior direito da página para criar uma cópia do repositório em sua conta.

2. **Clone o Repositório Forkado**
   - Abra o terminal.
   - Clone o repositório forkado para sua máquina local usando o comando:
     ```sh
     git clone https://github.com/seu-usuario/nome-do-repositorio.git
     ```
   - Substitua `seu-usuario` pelo seu nome de usuário do GitHub e `nome-do-repositorio` pelo nome do repositório.

3. **Crie uma Nova Branch**
   - Navegue até o diretório do repositório clonado:
     ```sh
     cd nome-do-repositorio
     ```
   - Crie uma nova branch para suas alterações:
     ```sh
     git checkout -b minha-nova-branch
     ```
   - Substitua `minha-nova-branch` por um nome descritivo para sua branch.

4. **Faça Suas Alterações**
   - Faça as alterações desejadas no código.
   - Certifique-se de seguir as diretrizes de estilo de código do projeto.

5. **Commit suas Alterações**
   - Adicione os arquivos modificados ao commit:
     ```sh
     git add .
     ```
   - Faça o commit das suas alterações com uma mensagem descritiva:
     ```sh
     git commit -m "Descrição das alterações"
     ```

6. **Envie suas Alterações para o Repositório Forkado**
   - Envie suas alterações para o GitHub:
     ```sh
     git push origin minha-nova-branch
     ```

7. **Abra um Pull Request**
   - Vá para a página principal do seu repositório forkado no GitHub.
   - Clique no botão "Compare & pull request" próximo à notificação sobre a sua branch recém-enviada.
   - Adicione um título e uma descrição detalhada para o seu pull request.
   - Clique em "Create pull request" para enviar suas alterações para revisão.

8. **Aguarde a Revisão**
   - Um dos mantenedores do projeto revisará seu pull request.
   - Esteja preparado para responder a perguntas e fazer alterações solicitadas.

</Steps>
